# BookStore Landing Page

## Descripción
Landing page para una tienda de libros en línea con carrito de compras, carrusel de productos y navegación completa.

## Características
- **Navbar**: Logo, menú de navegación, búsqueda, ayuda y carrito de compras
- **Carrusel de Productos**: Muestra 3 libros destacados con navegación
- **Modal de Detalles**: Vista detallada del producto con múltiples ángulos, precio, color, stock y opción de agregar al carrito
- **Footer**: Información de la empresa, enlaces a productos, redes sociales y contacto

## Estructura
- `index.html`: Página principal con configuración de Tailwind y React
- `app.js`: Componente principal con gestión del carrito
- `components/`: Componentes reutilizables (Navbar, ProductCarousel, ProductModal, Footer)
- `utils/`: Datos de productos

## Tecnologías
- React 18
- TailwindCSS
- Lucide Icons
